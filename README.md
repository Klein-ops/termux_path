Termux PATH Bridge

将 Termux 环境中的可执行命令桥接到 Android 系统 PATH，让你在任何终端（adb shell、MT 管理器终端等）都能直接调用 Termux 安装的程序。

特性

- 动态扫描：自动发现 Termux 中安装的所有命令，无需手动配置。
- 软链接架构：所有 wrapper 指向统一的主脚本，版本更新只需替换一个文件。
- 零残留：卸载 Termux 命令后，下次扫描自动清理对应 wrapper。
- 安全跳过：自动跳过系统已有命令和关键系统命令（如 su、mount），避免冲突。
- 权限修复：开机自动修复 Termux 目录权限，确保命令可执行。
- Magisk 规范：零硬编码路径，完全符合 Magisk 模块开发规范。
- POSIX 兼容：纯 POSIX sh 语法，任何 Android 设备均可运行。

安装

1. 下载模块的压缩包文件（通常以 .zip 结尾）。
2. 打开 Magisk Manager 应用。
3. 点击底部导航栏的「模块」图标进入模块管理页面。
4. 点击「从本地安装」按钮。
5. 在文件管理器中选择刚才下载的压缩包文件。
6. 等待刷入过程自动完成，然后根据提示重启手机。

重启后模块即会生效，无需额外配置。

使用方法

自动生效

重启手机后，模块会自动扫描 Termux 命令并生成 wrapper，之后即可在任意终端直接使用。

手动扫描（新增命令后）

当你在 Termux 中通过 pkg install 安装了新程序后，可以通过以下两种方式触发扫描更新：

方法一：Magisk Manager 一键执行

1. 打开 Magisk Manager，进入「模块」页面。
2. 找到名为「Termux PATH Bridge」的模块。
3. 点击模块底部的「操作」按钮。
4. 等待控制台输出扫描结果，即表示更新完成。

扫描完成后，新安装的命令即刻全局可用。
